# Playwright UI Test Suite

## Week 2 Plan

UI test automation using Playwright with TypeScript on SauceDemo (https://www.saucedemo.com) — a widely used QA practice application simulating an e-commerce storefront.

### Application Under Test
- **App:** SauceDemo [Demo Application]
- **URL:** https://www.saucedemo.com
- **Type:** E-commerce demo app (login, product listing, cart, checkout)

### Planned Work
- Identify test scenarios and write manual test cases first
- Set up Playwright project with TypeScript
- Implement Page Object Model (POM) structure
- Write automated UI tests covering key user flows:
  - Login (valid, invalid, locked user)
  - Product listing and sorting
  - Add to cart / remove from cart
  - Checkout flow end to end
- Integrate GitHub Actions CI to run tests on every push

### Tools
- Playwright with TypeScript
- GitHub Actions for CI
